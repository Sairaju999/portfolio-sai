# 🚀 Sai Raju Valluri | Portfolio

A modern, responsive, and interactive portfolio website built with **Vite**, **HTML5**, **CSS3**, and **JavaScript**. Featuring a sleek dark theme with neon cyan-to-purple gradients, smooth animations, and a beautiful glassmorphic design.

## 🌐 Live Demo
- [https://portfolio-sai-yuqt.onrender.com/](https://portfolio-sai-yuqt.onrender.com/)

## ✨ Features

- 🎨 **Modern Design** - Dark theme with neon gradients and glassmorphic components
- 📱 **Fully Responsive** - Works seamlessly on mobile, tablet, and desktop devices
- 🎭 **Light/Dark Theme Toggle** - Switch between dark and light themes
- ⚡ **Fast Performance** - Built with Vite for lightning-fast load times
- 🎬 **Smooth Animations** - Engaging transitions and hover effects
- 📝 **Resume Download** - One-click resume download functionality
- 🔗 **Social Links** - Direct links to GitHub, LinkedIn, and Email
- 📱 **Mobile Menu** - Responsive hamburger navigation
- 🎯 **Optimized SEO** - Meta tags and structured content

## 🚀 Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Sairaju999/portfolio-sai.git
   cd portfolio-sai
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```
   The site will be available at `https://portfolio-sai-yuqt.onrender.com/`

4. **Build for production**
   ```bash
   npm run build
   ```
   Production files will be in the `dist/` folder

5. **Preview production build**
   ```bash
   npm run preview
   ```

## 📝 Customization

### Edit Personal Information
- Update name in [index.html](index.html)
- Modify bio and descriptions in respective sections
- Add your social media links in the profile card

### Update Resume
- Replace `public/resume.pdf` with your resume file
- The download button will automatically link to the new file

### Customize Colors
Edit the CSS variables in [src/style.css](src/style.css):
```css
:root {
  --accent-1: #00f2fe;  /* Neon Cyan */
  --accent-2: #4facfe;  /* Neon Blue */
  --accent-3: #bd00ff;  /* Electric Purple */
  /* ... more variables ... */
}
```
